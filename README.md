Viper Scale | Miniaturas a escala

El proyecto trata de una página web para la compra de modelos a escala de aviones de combate, la plataforma es totalmente ficticia y no tiene intenciones de conseguir ganancias o proveer los productos mostrados en la página.
El proyecto ha sido realizado unicamente bajo la consigna del curso brindado por el programa "Talento Tech" enfocado en el diseño Front End.

El sitio web contiene todo lo incorporado en las clases del curso, especificandose puramente en el desarrollo Frontend:

-HTML semantico acompañado de estilos en css

-Manejo correcto de rutas de carpetas, subcarpetas, archivos.

-Uso de fuentes de google fonts para el texto.

-Elementos visuales como imágenes, cards, animaciones hover.

-Función agregada para simular un carrito de compras desarrollado en Javascript, con registro de productos en localStorage.

ATENCIÓN: Información como dirección de local, precios, personas, imagenes de locales fisicos y personas que aparecen son puramente ficticias son elementos puramente ficitios y desarrollados con Inteligencia Artifical. La información no es veridica y no se registra ningún tipo de compra.
